# AIニュース自動投稿アプリ

バズるニュース投稿を自動生成し、人間が最終判断して投稿するためのツール。

## 機能

### 1. ニュース取得・分析
- Google News RSS から日本語ニュース自動取得
- **Readability** を使用した本文抽出
- スコアリングロジックでバズりやすさを判定
  - 「終了」「廃止」「炎上」などショックキーワード: +6点
  - 「速報」「決算」などの高インテント: +5点
  - 本文長・記事の新鮮度もスコアに含める
  - 最小スコア（デフォルト: 4点）以上のニュースを選定

### 2. コンテンツ生成（すべてAI）

#### タイトルリライト
- 元のタイトルを 20〜40文字のインパクト重視版に変換
- 「え？」と思わせる表現を優先
- クリック率向上を狙う

#### 投稿案生成（X用）
3つのパターンからランダム選択：

**【shock型】**
```
【これヤバい】
〇〇がついに終了…

・理由 → ○○
・影響 → ○○

これ今後かなり影響出そう
```

**【summary型】**
```
〇〇が発表

これのポイント👇
・○○
・○○

◇関連
```

**【simple型】**
```
〇〇が話題に

理由は○○
今後は○○になりそう
```

#### 要約生成
- 100文字以内の要約
- なぜ話題なのか（2点）
- 今後どうなるか（2点）

### 3. 投稿機能

#### WordPress 自動投稿
- REST API 経由で直接投稿可能
- タイトル / 本文 / 抜粋 / ステータス自動設定
- og:image を HTML に埋め込み

#### X投稿案（手動）
- AI生成した投稿案をコンソール出力
- ユーザーが確認して手動投稿
- 自動投稿はしない（品質重視）

### 4. パフォーマンス最適化

#### キャッシュ機能
- `content_cache.json` に要約 / X投稿案 / リライトタイトルを保存
- 同じ記事は API 呼び出し不要（高速化）
- 最新 500 件を保持

#### 重複投稿防止
- `posted_urls.json` に投稿済み URL を記録
- RSS から同じニュースが来ても2重投稿しない
- 最新 3000 件を保持

#### 並列処理
- 複数のニュースを同時処理（要約 / 投稿案 / タイトル生成同時実行）

## インストール・セットアップ

```bash
npm install
cp .env.example .env
```

### .env ファイル設定

```ini
# OpenAI API キー（必須）
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4.1-mini

# WordPress 連携（オプション）
WP_BASE_URL=https://example.com
WP_USERNAME=admin
WP_APP_PASSWORD=xxxx xxxx xxxx
WP_STATUS=publish

# 実行制御
MAX_POSTS=3           # 1回の実行で処理する記事数
FETCH_LIMIT=20        # RSS取得時のスキャン対象数
MIN_SCORE=4           # 選定の最小スコア
SCHEDULE_MINUTES=0    # 定期実行（0=1回実行）
POSTED_STORE_FILE=posted_urls.json
```

## 使用方法

### 1回実行
```bash
node index.js
```

### 定期実行（1時間ごと）
```bash
SCHEDULE_MINUTES=60 node index.js
```

## 技術スタック

- **Node.js** - ランタイム
- **rss-parser** - RSS フィード取得
- **axios** - HTTP クライアント
- **@mozilla/readability** - 記事本文抽出
- **jsdom** - DOM パース
- **google-news-url-decoder** - Google News URL デコード
- **OpenAI API** - コンテンツ生成（GPT-4.1-mini）

## 注意点

- **X（旧 Twitter）への自動投稿は非対応** - 生成した投稿案を手動で投稿
- **WordPress が未設定の場合** - コンソール出力のみで投稿スキップ
- **API 呼び出し数** - キャッシュがあれば大きく削減可能
- **実行時間** - API 呼び出しのため初回は 30～60 秒程度
